# How To Access

1. open your terminal, access your project location. example :
   > cd miniclass-web/simple-graphql/
2. run this command below, then press ENTER :
   > npm run dev
3. Open
   > localhost:4000/graphql
4. Select all book :

```
{
 	movies{
        title
        genre
    }
}
```

5. select all actor :

```
{
    actors{
        name
        gender
    }
}
```
